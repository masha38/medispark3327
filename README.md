# 메디스파크 상업용 랜딩페이지

## 로컬 확인

정적 웹서버로 이 폴더를 실행한 뒤 `index.html`을 엽니다. 예: `python -m http.server 4173`

## Google Apps Script 연결

1. 고객 DB를 저장할 구글시트를 생성합니다.
2. Apps Script 프로젝트를 만들고 `Code.gs` 내용을 붙여넣습니다.
3. 프로젝트 설정 → 스크립트 속성에 `SPREADSHEET_ID`와 구글시트 ID를 등록합니다.
4. `setupSheet()`를 한 번 실행해 `방문상담_DB` 시트와 헤더를 만듭니다.
5. 웹 앱으로 배포하고 실행 사용자는 본인, 액세스 권한은 페이지 운영 환경에 맞게 설정합니다.
6. 배포 URL을 `config.js`의 `APPS_SCRIPT_URL`에 입력합니다.

상세 요청·응답 규격과 배포 절차는 `앱스크립트_명세서.md`, 시트 가져오기 샘플은 `google_sheet_columns.csv`를 참고하세요.

실제 고객 정보를 받기 전 개인정보처리방침, 보유기간, 상담 운영 주체와 Apps Script/시트 접근 권한을 확정하세요. URL이 비어 있으면 폼은 저장 실패를 안내하며 외부로 데이터를 보내지 않습니다.
